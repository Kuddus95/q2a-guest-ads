Q2A Guest Ads
======================

Plugin Version
--------------
- v1.5

This plugin will allow you to show advertisements to only guests or in the below listed spots. 
If you want to show ads only guest, then select 'registered' from dropdown menu for 'Hide This Ad for the below User Levels and Above'.  
You can use Google adsense or any HTML ad code. 

Features
--------
You can place ads at below 5 spots
- ad after menu navigation bar, just after horizontal line.
- ad before/after question, just before answers.
- ad after all answers, just before related questions.
- ad after first answer (will hide the ad after all answers in case only one answer is there)
- ad before/after all questions.
- ad in sidebar.
- ad on left end of screen on large screens.

Installation Guide
------------------
- Extract zip file.
- Place directory called `q2a-guest-ads` in qa-plugin folder.
- Done

Minimum Question2Answer Version Required
----------------------------------------
- v1.6 (works with v1.8.5)

Admin Guide
-----------
- Login with admin id.
- Go to Admin > Plugins page.
- Scroll down to "Q2A Guest Ads" in plugin list.
- Click on "options" to enable ads spots and place ad code.
